# Hyprland configuration
#
# Files in this directory are managed by Home Manager (home/default.nix).
# They are symlinked to ~/.config/hypr/ on the active machine.
#
# Typical contents:
#   hyprland.conf        — main Hyprland config
#   hyprpaper.conf       — wallpaper config (if using hyprpaper)
#   keybinds.conf        — keybinding definitions (can be split off for clarity)
#   monitors.conf        — per-machine monitor layout (override in hosts/ if needed)
#   rules.conf           — window rules
