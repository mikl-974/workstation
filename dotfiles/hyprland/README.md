# Hyprland configuration
#
# Files in this directory are managed by Home Manager (home/roles/*.nix, home/users/*.nix, or home/targets/*.nix).
# They are symlinked to ~/.config/hypr/ on the active machine.
#
# Active contents:
#   hyprland.conf        — main Hyprland config (autostart + keybindings + base UX)
#
# Future optional contents:
#   hyprpaper.conf       — wallpaper config (if using hyprpaper)
#   keybinds.conf        — keybinding definitions (split only if the main file becomes too large)
#   monitors.conf        — per-machine monitor layout (override in hosts/ if needed)
#   rules.conf           — window rules
